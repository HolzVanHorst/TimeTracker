# TimeTracker

Automatisches App-Usage-Tracking für Windows.

## Features
- Mehrere Apps gleichzeitig tracken
- Automatische Start/Stop-Erkennung
- SQLite-Datenspeicherung
- Statistiken (Tag/Woche/Monat)
- Autostart-Integration

## Installation
pip install -r requirements.txt
python -m timetracker

## Verwendung
Starten:
python -m timetracker

Menu:
1. Initialisierung (Apps hinzufügen)
2. Tracking starten
3. Statistiken ansehen
4. Settings
